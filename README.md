# <i style="color: #6495ED;">BurgerShop</i>  Burger Sipariş Projesi

Bu proje, ASP.NET Core MVC, Entity Framework, MSSQL, Bootstrap, CSS, HTML ve Identity teknolojilerini kullanarak, verilen 1 haftalık proje süresi içerisinde üç kişilik grup olarak oluşturmuş olduğumuz bir burger sipariş projesidir.

## Kullanım
- Kayıt olabilir ve giriş yapabilirsiniz.
- Kayıt sırasında mail doğrulaması yapmaktadır.
- Giriş yaptıktan sonra sepete sipariş ekleyebilir ve sipariş oluşturabilirsiniz.
- Admin olarak giriş yapabilir ve menü, hamburger ve yan ürünleri oluşturabilir, güncelleyebilir ve silebilirsiniz.
